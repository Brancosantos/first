1. Considere a tabela “alunos” com a seguinte estrutura: CREATE TABLE alunos (
Id_aluno INT PRIMARY KEY,
nome-aluno VARCHAR(255),
data_nascimento DATE,
cidade_natal VARCHAR(100)
);
Suponha que você deseja encontrar a idade média dos alunos na cidade de São Paulo. Qual consulta SQL você usaria?

R. SELECT AVG(YEAR(NOW()) – YEAR(data_nascimento) FROM alunos WHERE cidade_natal = ‘São Paulo’;

2.Considere a tabela “disciplina” com a seguinte estrutura:

CREATE TABLE disciplina (

Id_disciplina INT PRIMARY KEY,
Nome_disciplina VARCHAR(255)
);
Suponha que você deseja listar todas as disciplinas cujo nome começa com a letra “C”. Qual a consulta SQL você usaria?
R. SELECT * FROM disciplina WHERE nome_disciplina LIKE “C%”

3. Suponha que você deseja calcular a média das notas dos alunos que nasceram em São Paulo usando a tabela “alunos”. Qual consulta SQL você usaria para isso



R. SELEC AVG(nota) FROM notas WHERE id_aluno IN (SELECT id_aluno FROM alunos WHERE cidade_natal = ‘São Paulo’):
